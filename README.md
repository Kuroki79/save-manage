# save-manage (存档管理器)

> 一个存档管理工具，支持备份和还原，还计划添加更多功能。

## 功能
- [x] 指定存档目录的备份和还原
- [x] 软件配置的导入与导出
- [ ] 云备份功能
- [x] 导入Steam平台的游戏
- [x] 暂时就想到这些...
